# Обучение LoRA для NeuroMango

Этот раздел содержит всё необходимое для тонкой настройки (Fine-Tuning) вашей собственной модели Llama-3, чтобы она идеально отыгрывала роль Неманго. Мы будем использовать Google Colab (бесплатно) и библиотеку Unsloth.

## Шаг 1: Генерация Датасета
Прежде чем обучать модель, ей нужно показать примеры диалогов. Я написал скрипт, который сам придумает сотни примеров на основе её характера (`core_memory.txt`).

1. Откройте терминал на вашем ПК в папке `NeuroMango`.
2. Убедитесь, что у вас прописаны ключи API (Gemini или локальная модель) в файле `.env`.
3. Запустите скрипт:
   ```bash
   python colab/generate_dataset.py
   ```
4. Скрипт создаст файл `dataset.jsonl` в корневой папке. Это и есть учебник для нейросети!

## Шаг 2: Обучение в Google Colab
Вам не нужна мощная видеокарта для обучения — мы арендуем её у Google бесплатно!

1. Откройте сайт [Google Colab](https://colab.research.google.com/).
2. Нажмите **Файл -> Загрузить блокнот (Upload notebook)**.
3. Выберите файл `colab/Train_NeuroMango_LoRA.ipynb` из этой папки.
4. В верхнем меню выберите **Среда выполнения -> Сменить среду выполнения**. Выберите **T4 GPU**.
5. Нажмите на значок папки 📁 в левом меню Colab. Перетащите туда ваш сгенерированный файл `dataset.jsonl`.
6. Выполняйте ячейки кода по очереди (кнопка Play слева от ячейки).

## Шаг 3: Как использовать обученную модель
После того как Colab закончит обучение (последняя ячейка), он сохранит файл в папку `model/`.
1. Скачайте файл `model-unsloth-Q4_K_M.gguf` к себе на компьютер (на тот ПК, где у вас RTX 5060).
2. Закиньте этот файл в LM Studio (просто перетащите его в программу или положите в её папку с моделями).
3. Загрузите эту модель в LM Studio, запустите сервер.
4. НейроМанго готова к стриму! Теперь она будет отвечать моментально и в точности так, как вы её обучили, даже без длинного системного промпта!
