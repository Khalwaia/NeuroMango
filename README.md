# 🥭 NeuroMango (AI Streamer & Assistant)

NeuroMango — это продвинутый локальный ИИ-ассистент и виртуальный стример (VTuber). Она обладает "мозгом" для общения, долгосрочной памятью, способностью спонтанно мыслить (Heartbeat), ставить музыку, читать чат Twitch и реагировать на донаты.

---

## 🧠 Архитектура (Под капотом)

Проект использует гибридную архитектуру, совмещая локальные скрипты с мощными облачными LLM:

1. **Основной Мозг (Brain):** `deepseek/deepseek-v4-flash` (через OpenRouter). Отвечает за генерацию ответов, эмоций, управление музыкой и системой.
2. **Сердцебиение (Heartbeat):** `openai/gpt-4o-mini` (через OpenRouter). Фоновый процесс, который позволяет НеМанго "думать" и действовать самостоятельно, даже когда к ней никто не обращается.
3. **Память (Memory):**
   * **База данных:** Векторная БД `ChromaDB` (полностью локально).
   * **Эмбеддинги:** `paraphrase-multilingual-MiniLM-L12-v2` — переводит текст в векторы для быстрого поиска по памяти.
   * **Свинопас (Экстрактор):** `llama-3.1-8b-instant` (через Groq API). Фоновый процесс, который вытягивает важные факты из диалога и сохраняет их в ChromaDB.
4. **Генерация голоса (TTS):** Локальная модель `XTTS-v2` от Coqui с параметрическими эмоциями и правильной расстановкой ударений (`ruaccent`).
5. **Музыка:** Ищет музыку на YouTube (через `yt-dlp`) и стримит её напрямую через кастомный виджет для OBS.

## 🛠️ Установка

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Khalwaia/NeuroMango.git
   cd NeuroMango
   ```
2. Создайте виртуальное окружение и установите зависимости:
   * Вы можете просто запустить скрипт `install_env.bat` (он сделает всё сам).
   * Или вручную:
     ```bash
     python -m venv venv
     venv\Scripts\activate
     pip install -r requirements.txt
     ```

## ⚙️ Настройка (Ключи API)

Для работы мозга и памяти нужны ключи API.
В корневой папке проекта создайте файл `.env` (можете скопировать из `.env.example`, если он есть) и заполните его:

```env
# API Ключ для OpenRouter (для DeepSeek и GPT-4o-mini)
OPENROUTER_API_KEY="sk-or-v1-ваш-ключ-от-openrouter"
LLM_API_KEY="sk-or-v1-ваш-ключ-от-openrouter"

# URL и Модель для основного общения
LLM_BASE_URL="https://openrouter.ai/api/v1"
LLM_MODEL="deepseek/deepseek-v4-flash"

# API Ключ для Groq (Свинопас - экстрактор памяти)
GROQ_API_KEY="gsk_ваш-ключ-от-groq"

# Twitch (для чтения чата)
TWITCH_TOKEN="oauth:ваш-oauth-токен-twitch"

# DonationAlerts (для реакции на донаты)
DA_WIDGET_TOKEN="ваш-токен-от-donation-alerts"
```

> **Важно:** Файл `.env` скрыт в `.gitignore`, поэтому ваши ключи никогда не попадут в публичный доступ на GitHub!

## 🚀 Запуск

Для старта сервера просто запустите:
```bash
run_server.bat
```
*(Или вручную из консоли: `venv\Scripts\activate` -> `python server.py`)*

При первом запуске локальные модели (TTS и Эмбеддинги) могут начать скачиваться. Это займет некоторое время и потребует пары гигабайт на диске.

## 🎬 Подключение к OBS

В проекте есть несколько веб-виджетов, которые нужно добавить в OBS как **"Источник браузера" (Browser Source)**.

### 1. Музыкальный виджет
Отображает текущий трек, который НеМанго включила, а также играет саму музыку.
* **URL:** `http://127.0.0.1:8766/static/widget.html`
* **Ширина/Высота:** По вкусу (например, 400x150).
* **ВАЖНО:** Обязательно поставьте галочку **"Управление аудио через OBS" (Route audio to OBS)**, чтобы вы могли регулировать громкость музыки прямо в микшере OBS!

### 2. Виджет DonationAlerts
Перехватывает донаты для обработки в мозге НеМанго (чтобы она зачитывала и комментировала их).
* **Локальный файл:** Выберите `da_widget.html` (находится в корне проекта).
* Этот виджет работает в фоне и передает донаты на локальный сервер Python.

## 📂 Структура проекта (кратко)

* `server.py` — Главный файл, поднимает FastAPI сервер и WebSockets.
* `llm_engine.py` — Работа с ИИ, генерация ответов, Heartbeat.
* `memory_manager.py` — Векторная база данных ChromaDB (память).
* `tts_engine.py` — Генерация голоса XTTS.
* `music_service.py` — Поиск и логика очереди воспроизведения музыки.
* `action_handler.py` — Исполнение команд (например, `[Action: play_music...]`).
* `core_memory.txt` — Системный промпт (личность и характер НеМанго).
* `config.py` — Основные настройки (пути, порты, дефолтные модели).
